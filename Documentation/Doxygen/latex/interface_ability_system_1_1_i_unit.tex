\doxysubsection{Detailed Description}
Composite interface representing a fully functional gameplay unit. ~\newline
 Combines grid movement, health management, status effects, and buff functionality. ~\newline
~\newline
. 

Implementers must provide :\+ ~\newline
 {\bfseries{-\/ ability\+Instances :\+}} A list of runtime ability instances available to this unit. ~\newline
~\newline


{\bfseries{Note for Developers :\+}} ~\newline
 This is the central interface to update when expanding unit‑level functionality. ~\newline
 Any new system that all units must support should be added here. ~\newline
~\newline
