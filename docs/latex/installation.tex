\hypertarget{installation_autotoc_md45}{}\doxysubsubsection{\texorpdfstring{Manual Install}{Manual Install}}\label{installation_autotoc_md45}
Manual installation is also possible if the Package Manager is unavailable or not preferred. ~\newline
 To install the package manually, download and extract it, then place the extracted folder inside your project\textquotesingle{}s {\ttfamily Packages/\+} directory (located in the project root). If the {\ttfamily Packages/\+} folder does not exist, create it manually.

Ensure that you copy only the package’s root folder — the one that directly contains the {\ttfamily package.\+json} file.\hypertarget{installation_autotoc_md46}{}\doxysubsection{\texorpdfstring{Verification}{Verification}}\label{installation_autotoc_md46}
After installing both, you should be able to find both {\ttfamily Modular Architecture} and {\ttfamily Ability System} in your project\textquotesingle{}s browser in the package/\+ section. If not, check to see if you have packages enabled in your project browser, and restart Unity.\hypertarget{installation_autotoc_md47}{}\doxysubsection{\texorpdfstring{Thank You For Reading}{Thank You For Reading}}\label{installation_autotoc_md47}
