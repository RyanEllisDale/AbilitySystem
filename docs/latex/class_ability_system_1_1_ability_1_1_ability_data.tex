\doxysubsection{Detailed Description}
Data asset defining an ability within the \doxylink{namespace_ability_system_1_1_ability}{Ability} System. Stores all permanent configuration such as effects, audio, categorization, requirements, and cooldown settings. ~\newline
~\newline
. 

Contains :\+ ~\newline
 {\bfseries{-\/ id :\+}} A unique string identifier for referencing this ability, ~\newline
 {\bfseries{-\/ audio\+Clips :\+}} A list of audio clips to be referenced by developers ( via event calls or reading data ), ~\newline
 {\bfseries{-\/ plug\+In\+Effects :\+}} A list of \doxylink{class_ability_system_1_1_ability_1_1_effect}{Effect} objects executed when the ability is used , ~\newline
 {\bfseries{-\/ range :\+}} The maximum distance at which the ability can target ( must be implemented ny developers for their own range system ), ~\newline
 {\bfseries{-\/ damage\+Type :\+}} The type of damage this ability deals, ~\newline
 {\bfseries{-\/ ability\+Category :\+}} The ability category this ability belongs to, ~\newline
 {\bfseries{-\/ conditions :\+}} Requirements that must be met before the ability can be used, ~\newline
 {\bfseries{-\/ supplies :\+}} Resource costs required to activate the ability, ~\newline
 {\bfseries{-\/ turn\+Cooldown :\+}} The number of turns required before the ability can be reused. ~\newline
