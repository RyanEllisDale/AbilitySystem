This chapter explains:\+ \begin{DoxyVerb}- What an AbilityData is

\end{DoxyVerb}
\hypertarget{ability_page_autotoc_md6}{}\doxysubsubsection{\texorpdfstring{1. Understanding Abilities}{1. Understanding Abilities}}\label{ability_page_autotoc_md6}
Abilities in this system are split into two layers:\+ Ability\+Data (Permanent Data)

A Scriptable\+Object that defines:\+ \begin{DoxyVerb}- Effects
- Conditions
- Supplies (resource costs)
- Cooldown
- Range
- Categorization
- Audio clips
- Damage type
\end{DoxyVerb}


This is the design-\/time representation of an ability. Ability\+Instance (Runtime State)

A lightweight class that stores:\+ \begin{DoxyVerb}- A reference to the AbilityData
- The current cooldown
\end{DoxyVerb}


Each unit holds a list of Ability\+Instances.\hypertarget{ability_page_autotoc_md7}{}\doxysubsubsection{\texorpdfstring{2. Creating an Ability}{2. Creating an Ability}}\label{ability_page_autotoc_md7}
Create a new ability via:\+ Code

Right‑click → Create → Abilities → Create New Ability

This generates an Ability\+Data asset.\hypertarget{ability_page_autotoc_md8}{}\doxysubsubsubsection{\texorpdfstring{Ability Fields}{Ability Fields}}\label{ability_page_autotoc_md8}

\begin{DoxyItemize}
\item id A unique string identifier for referencing the ability.
\item audio\+Clips Optional audio clips developers can use when the ability is executed.
\item plug\+In\+Effects A list of Effect Scriptable\+Objects that define what the ability does.
\item Effects are executed in order when the ability is activated.
\item damage\+Type /\+ ability\+Category User-\/defined enums for classification.
\item range a simple integer value representing maximum targeting distance.
\end{DoxyItemize}

\begin{DoxyVerb}- Keep abilities small and composable
- Use multiple Effects instead of one large one
- Use Conditions to restrict ability usage
- Use Supplies to enforce resource costs
- Use DataReferences for dynamic cooldowns or scaling values
- Keep targeting logic outside the ability system
- Use events to trigger UI, VFX, and SFX
\end{DoxyVerb}
