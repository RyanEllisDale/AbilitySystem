\hypertarget{api_page_autotoc_md49}{}\doxysubsection{\texorpdfstring{How‑\+To:\+ Using the Ability System API}{How‑\+To:\+ Using the Ability System API}}\label{api_page_autotoc_md49}
The Ability\+System\+API is the central access point for interacting with the Tactics Ability System at runtime. It provides a clean, unified interface for:\+ \begin{DoxyVerb}- Activating abilities
- Checking resource requirements
- Applying and removing statuses
- Applying and removing buffs
- Processing turn‑based logic
- Dispatching global events
\end{DoxyVerb}


This chapter explains:\+ \begin{DoxyVerb}- What the API does
- How abilities are executed
- How supplies are evaluated
- How statuses and buffs are applied
- How turn start / turn end logic works
- All available events
- Best practices
\end{DoxyVerb}
\hypertarget{api_page_autotoc_md50}{}\doxysubsubsection{\texorpdfstring{1. Understanding the Ability\+System\+API}{1. Understanding the Ability\+System\+API}}\label{api_page_autotoc_md50}
The API is a static class that exposes all high‑level operations. It is designed to be:\+ \begin{DoxyVerb}- Simple — one‑line calls for common actions
- Consistent — same pattern for buffs, statuses, and abilities
- Decoupled — units only need to implement IUnit, IStatusContainer, and/or IBuffContainer
- Event‑driven — UI and VFX should subscribe to API events
\end{DoxyVerb}


The API does not store state. It orchestrates the systems that do. \hypertarget{api_page_autotoc_md51}{}\doxysubsubsection{\texorpdfstring{2. Ability Activation}{2. Ability Activation}}\label{api_page_autotoc_md51}

\hypertarget{api_page_autotoc_md59}{}\doxysubsubsection{\texorpdfstring{10. Best Practices}{10. Best Practices}}\label{api_page_autotoc_md59}
\begin{DoxyVerb}- Use the API for all ability, buff, and status interactions
- Never modify BuffManager or StatusManager directly
- Use events for UI and VFX
- Keep turn logic centralized
- Keep abilities stateless; let the API handle execution
- Keep buffs and statuses backend‑only
- Use DataReferences for dynamic resource values
\end{DoxyVerb}
