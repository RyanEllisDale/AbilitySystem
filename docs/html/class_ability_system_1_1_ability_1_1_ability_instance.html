Detailed Description

Runtime instance of an AbilityData asset. Stores temporary state such as cooldown and provides a reference to the permanent ability configuration.

.

Contains :
- ability : The permanent AbilityData asset this instance is based on,
- currentCooldown : The number of turns remaining before the ability can be used again.
