 CBuffDataData asset for buffs, contains all the permenant data for buffs, and lives within the project's files.
Contains the list of modifier rules which determine how buffs interact with the stats of IBuff references.

Contains :
- id : A unique string identifier for referencing this buff,
- description : A multiline description used for UI, debugging, or doccumentation,
- duration : How many turns the buff remains active once applied,
- modifiers : The array of StatModifier rules that define how this buff affects stats.
 CBuffInstanceA runtime instance for buff data to be applied to buff managers, modifies the stats of IBuffContainer containers based on the data's modifiers.
Contains the inpermenant runtime data for active buffs / de-buffs that are applied to a target / buff-manager.

Contains :
- Data : The permenant data asset file for the instance to read from,
- CurrentDuration : How many turns the Instance has left before being removed.

Responsible for:
- Triggering Modification : When a BuffInstance is applied it will trigger all the stat modifications in it's data,
- Expiring Modification : When a BuffInstance's duration ends, it will trigger the inverse of stat modifications in it's data,
- Storing Current Duration : As each turn passes the API will reduce it's duration and remove the Instance when it hits 0.
 CBuffManagerThe Buff Manager is a composite runtime container that manages all active BuffInstance objects applied to a target implementing IBuffContainer.
This class is a the Buff Functionality mirror to Status Manager.

Responsible for:
- Adding Buffs : Creates new runtime instances and applies their stat modifiers,
- Removing Buffs : Expires instances and reverses their stat modifiers,
- Tracking Active Buffs : Maintains a list of all currently applied BuffInstances.
 CIBuffContainerInterface for any object capable of receiving and managing buffs.

- GetStat : Retrieves a stat value (float) by its string identifier,
- ModifyStat : Applies additive or multiplicative changes to a stat.
